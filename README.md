# antimass-feedback

Welcome to the public roadmap and issue tracker for **Antimass Labs**.

To maintain our zero-bloat architecture and prevent spam clones, the core MockEngine codebase remains private. However, we build our roadmap entirely in public. We listen to our users, not algorithms.

### How to use this repository

Keep it lightweight. If you have feedback, open a New Issue and prefix your title with one of the following tags:

* **🐛 Found a bug?** Tag it `[BUG]` and include your browser and steps to reproduce.
* **⚡ Want a new tool?** Tag it `[FEATURE]` and briefly explain the engineering friction it solves.
* **📚 Need a tutorial or blog post?** Tag it `[TUTORIAL]` and tell us what technical concept, architecture, or workflow you want us to break down next.
* **👍 Support an existing idea?** Please don't clutter threads with "+1" comments. Just add a 👍 reaction to the original issue to vote it up our priority list.
